inBloom Learning Map Authoring Tool User Guide

Getting Started with inBloom Learning Map Authoring Tool

  • welcome

    Welcome to inBloom Learning Map Authoring Tool. inBloom Learning Map Authoring Tool is a tool for viewing, editing, and creating standards and then adding those standards into a path.

  • main screen

    This is the main screen for the inBloom Learning Map Authoring Tool. This window contains everything necessary for creating, viewing, and publishing Standards and Paths.

About Standards in inBloom Learning Map Authoring Tool

  • Common Core State Standards

    These are official standards which are being adopted by most states. They are separated into Mathematics and ELA-Literacy. You can learn more about them at corestandards.org. These standards can act as a template for designing standards heirarchies.


    This is where you will find the Common Core State Standards. Clicking the title will expand the section.


    You will then select the Subject.


    On this bar at the top, you can select which grades you wish to see from the Common Core State Standards.


    You will then be able to expand the elements within the subject you chose.


    You can expand any element which has an arrow to the left of the name by simply clicking on the name.


    When you get to the lowest level, there should be text associated with the element. The text is viewable by clicking the Info (i) button.


    Clicking the Info button will bring up the Info box where you will find the name and description of the standard as well as a button for showing resources.


    Clicking the resources button will bring up the Resources box where a list of links to related Resources will be displayed.


    You can click the checkbox on an element in the Standards list to select that element and all of the elements inside it.


    When you have checked some standards, you can press the Add Path button to add those elements to the Path Window.


    With checked standards, you could also add those standards to your Draft section for editing.

  • Draft (Personal Learning Map)

    These are your standards that you can edit, download, and publish. To get standards into the Personal section, you can either duplicate standards from the Public or Common Core sections, upload a JSON file that has standards information in the expected format, or create a framework from scratch with the provided buttons.


    This is where you will find the personal draft standards. Clicking the title will expand the section.


    The standards that we copied from the Common Core must be abbreviated to fit into the Standards List. Hovering will reveal the full name.


    In the Draft section, you will be able to drag standards around to place them in different parts of the tree. You can also drag standards onto other standards to put the dragged standard inside of the other standard.


    You can still see the Info on Standards in the Draft section. But instead of resources, you have the ability to edit the standard.


    After hitting the 'edit' button, the name and description of the standard will appear in input fields. You may now change the text of either field.


    The 'edit' button was replaced with a 'save' button. Pressing the 'save' button saves the name and description.


    The down arrow button allows you to download a set of standards that you've created or edited.


    Once you press the Download button, a file named "download" will be saved by your browser. It will save to your browser's default download directory.


    You can later upload these files using the upload widget indicated in the picture above.


    If you have checked standards in your Draft section, you can delete them by pressing the 'Delete' button.


    You can also create new standards underneath the default "New Framework" element in the Draft section. Simply press the 'Create element' button.


    The standards will be named 'new standard' by default.


    The up arrow button allows you to publish a set of standards that you've created or edited. Publishing a set of standards will send it to the Learning Registry Index.


    An alert box will pop-up to tell you that you are now publishing standards.

  • Published Public Standards

    These are standards that others have published to the LRI. They are now available for anyone to use.


    This is where you will find the Published Public Standards. Clicking the title will expand the section. The Published Public Standards section includes all of the standards that have been published to the Learning Registry Index. You must click the set of standards you wish to see before it loads.


    Once you have clicked the set of standards you wish to see, it will load and have an arrow and a checkbox.

About Paths in inBloom Learning Map Authoring Tool

  • Paths

    Paths are a specific order to go through a set of standards. A Path should be like a helpful outline for a lesson plan for educators. One standard leads to the next. In a Path, standards can be grouped together. The meaning of a group within a Path is that the standards within the group can be done in any order, but they must all be completed before moving on to the next standard.

  • The Path Window

    This is the Path Window. It has been pre-loaded with some standards from the Common Core.


    In the image above, the Connection Tool is highlighted on the toolbar. The Connection Tool allows you to create an arrow on the map which connects one standard node to another node. First, you must click the node where the arrow should start. The node will change color to indicate that it has been selected.


    With one node selected, clicking a second node will create the connection between the two nodes.


    It is possible to group nodes within the Path. This is done by simply clicking a node and dragging it to another node.


    Here is an example of a Path. It contains a few groups and all of the nodes and groups are connected by arrows.


    The image above shows the Disconnection Tool highlighted on the toolbar. The Disconnection Tool allows you to break a connection between nodes. Simply click on the arrow that you wish to remove.


    The Open Path tool allows you to open a previously saved Path file or open a Path which has been published to the Learning Registry Index.


    The Save Path tool allows you to save the current Path to a file or publish the current Path to the Learning Registry Index.


    The Delete Path tool will clear the current Path of all nodes and connections.


    The Print Path tool will start the process of printing your current Path to a printer.


    The Style Path tool allows you to change the background image and the colors of nodes.
